# 🏆 Campeonato de Pesca

Aplicação web para gerenciar um campeonato de pesca, com frontend em **GitHub Pages** e backend em **Flask (Render)**.  
Permite adicionar competidores, registrar os tamanhos dos peixes e gerar automaticamente o ranking e o troféu para o pescador que pegou o maior peixe.

---

## 🚀 Funcionalidades
- Adicionar competidores, e adicionar 4 tamanhos de peixes e separar os tamanhos por virgula.
- Exibir lista de competidores adicionados.
- Enviar dados para o backend e receber o resultado da competição.
- Mostrar ranking em tabela estilizada com destaque para os três primeiros (ouro, prata e bronze).
- Exibir troféu para o maior peixe capturado.

---

## 🖥️ Tecnologias utilizadas
- **Frontend:** HTML, CSS, JavaScript (hospedado no GitHub Pages).
- **Backend:** Flask + Flask-CORS (hospedado no Render).
- **Deploy:** GitHub Pages para o site e Render para a API.

---

## 📂 Estrutura do projeto
- `index.html` → página principal com formulário, tabela e estilos.
- `style` embutido → cores, gradientes e destaques visuais.
- `script` → lógica para adicionar competidores e enviar dados ao backend.
